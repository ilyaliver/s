# s

Это браузерная интерактивная страница «Поиск сигнальных прыжков» — мини-игра в стиле gacha/синулятор с анимированной космической тематикой.

## Что умеет проект

- интерактивный интерфейс с космической тематикой;
- система «поиска» с pity-ограничениями и гарантированными результатами;
- инвентарь полученных предметов;
- магазин валюты/факелов;
- сохранение прогресса в локальном хранилище браузера.

## Файлы проекта

- `index.html` — основной интерфейс и логика приложения;
- `music.mp3` — фоновой аудиотрек;
- `LICENSE` — лицензия проекта (Apache License 2.0);
- `README.md` — описание проекта.

## Как открыть

Проект можно открыть напрямую в браузере:

- откройте файл `index.html`;
- или разверните репозиторий на GitHub Pages / локальном HTTP-сервере.

### Локальный запуск

Если нужен локальный сервер, можно запустить:

```bash
python3 -m http.server 8000
```

После этого откройте в браузере:

```text
http://localhost:8000
```
