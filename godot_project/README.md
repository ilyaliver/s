# Поиск Сигнальных Прыжков - Godot 4.x

Адаптация браузерной интерактивной игры "Поиск сигнальных прыжков" для игрового движка Godot 4.x.

## Особенности

- Полная адаптация механик гача-симулятора
- Альбомная ориентация (1280x720)
- Космическая тематика с анимированным фоном
- Система pity-гарантов (EX: 90, S: 50)
- Инвентарь с системой созвездий (C1-C5) и уровней (1-25)
- Магазин валюты
- Сохранение прогресса
- Фоновая музыка

## Запуск

1. Откройте Godot Editor 4.x
2. Импортируйте проект из папки `godot_project`
3. Нажмите Play для запуска

## Механики

### Ранги предметов
- EX (6 звезд) - 0.1% базовый шанс
- SSS (5 звезд) - 0.4%
- SS (5 звезд) - 1.0%
- S+ (5 звезд) - аномалия при потере 50/50
- S (5 звезд) - 2.5%
- A (4 звезды) - 6.0%
- B (3 звезды) - 10.0%
- C (3 звезды) - 15.0%
- D (2 звезды) - 20.0%
- E (2 звезды) - 20.0%
- F (1 звезда) - 25.0%

### Pity система
- EX гарантия: каждые 90 поисков
- S гарантия: каждые 50 поисков
- При потере 50/50 на EX следующий EX гарантирован

### Система созвездий
- C1-C5: повышение уровня созвездия каждые 25 уровней
- Максимальный дубликат конвертируется в 1600 кристаллов
